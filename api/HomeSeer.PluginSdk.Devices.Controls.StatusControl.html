    Class StatusControl

    A control associated with a feature on a HomeSeer system. Formerly referred to as a VSPair.

    This defines a control that will be available for a user to interact with.

    Remarks

    Legacy VSPairs used to be able to be defined as either status-only, control-only, or both, but this is no longer allowed. All StatusControls are considered both, and a StatusGraphic will override the Status on the feature if it is configured for the feature's current Value.

    If you are looking to add a status-only control to a feature, create a StatusGraphic instead.
